# DateSorter

<h3>To launch app:</h3>
- pull this project <br>
- Run this app in IDEA (you can pass your data in method main) <br>

<h3>Project description:</h3>
Method sortDates sorts list of LocalDate according to specific condition: <br>
to cut the long story short, it works as natural order sorter with dates <br>
which months include 'r' and works vice versa with other ones. <br>


<h3>In this APP were used such technologies like:</h3>
- org.apache.maven, version 4.0.0 <br>
- java, version 17 <br>
- junit 4 <br>
